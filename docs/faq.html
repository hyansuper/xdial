常见问题

如何重置 WiFi？

设备启动时若无法连接 WiFi，会自动进入 WiFi 配网状态。

您也可以手动让设备进入 WiFi 配网状态：

  1. 在设备上电前按住左键，并在启动时保持按下状态约1秒直到“嘀”的一声。
  2. 根据提示连接设备提供的 WiFi 热点进行设置。

无法访问设置页面

  1. 请检查设备已联网且与您的手机处于同一局域网内。
  2. 进入设备的设置界面，点击右键可开启/关闭设置页面。确保其为开启状态。
  3. 扫描设备屏幕上的二维码或访问其下方的网址便可进入设置页面。

没有天气数据

天气数据是通过免费订阅和风天气获取的，步骤如下：

  1. 注册和风天气的开发账号。
  2. 登录项目管理页面 > 点击创建项目 > 选择免费订阅和Web API > 创建 > 查看并复制 KEY。
  3. 转到设置页面，在天气里填入 KEY，查找并选择您所在城市，然后点击保存。
  4. 进入设备的天气界面查看，若仍然没有数据，可点击右键刷新，根据对应的错误状态码排查原因。

下载自选表盘后设备不断重启

表盘下载过程出错或表盘文件本身格式错误都可能导致设备在加载表盘时出错而重启。

您可以尝试让设备启动时跳过表盘加载，然后进入设置页面删除自选表盘：

  1. 在设备上电前按住右键，并在启动时保持按下状态约1秒直到“嘀”的一声。
  2. 若设备能顺利启动，您可以在设置页面 > 表盘 > (自选表盘) > 点击删除按钮来删除自选表盘。
  3. 若问题仍未解决，您可以尝试恢复出厂设置。

如何恢复出厂设置？

您可以在设备启动阶段恢复出厂设置，这可能有助于解决一些未知问题：

  1. 在设备上电前同时按住左键和右键，并在启动时保持按下状态约1秒直到“嘀嘀”两声。
  2. 此时设备将删除设置页面里的所有个性化设置（包括和风天气的 KEY 和自选表盘），但保留 WiFi 设置。
