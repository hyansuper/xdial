天气

天气数据是通过 免费订阅 和风天气 获取的，步骤如下：

  1. 注册 和风天气的开发账号。

  2. 登录和风天气 项目管理 页面 → 点击 创建项目 → 任意填写名称，选择 免费订阅 → 点击 创建 → 创建凭据 → 选择 API KEY → 任意填写名称，点击 创建 → 复制生成的 API KEY。

  3. 返回网页版设置页面，在 “天气” 栏里填入 KEY，查找并选择您所在城市，然后点击 “保存” 按钮 。

  4. 此时您便可以在设备的天气界面看到气象数据，设备会根据设置定期更新数据。

备注

设备每天的天气数据请求次数约为 ((60 / 刷新间隔 + 3) * 24) 次，而目前免费订阅账号的数据请求 限制 是 1000 次/天。
因此，如果刷新间隔都设为 5 分钟，则两个设备可以共享一个 KEY; 如果都设为 10 分钟，则 4 个设备可以共用一个 KEY。
